# Convex hull

```julia
using LazySets
using Plots
```

```julia
points(N) = [randn(2) for i in 1:N]
v = points(30)
hull = convex_hull(v)
p = plot([Singleton(vi) for vi in v])
plot!(p, VPolygon(hull), alpha=0.2)
```
